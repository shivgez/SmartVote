# SmartVote - Voting System

## Description
SmartVote is a simple voting system built using Java and MySQL.  
It allows students to vote securely and prevents duplicate voting.

---

## Technologies Used
- Java
- MySQL
- JDBC

---

## Features
- Student Login using ID and Password
- One student can vote only once
- Admin Panel
- Add Candidates
- View Results

---

## How It Works
1. Students login using ID and password
2. They choose a candidate and vote
3. Votes are stored in database
4. Admin can view results and manage candidates
